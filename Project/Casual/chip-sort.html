Giới thiệu dự án

Chip Sort là game casual vui nhộn với gameplay chính là chọn các màu chip giống nhau di chuyển cùng 1 ô. Người chơi cần sắp xếp các chip một cách logic để hoàn thành các level.

Game được phát triển trong 1 tháng, tập trung vào trải nghiệm chơi đơn giản nhưng đầy thách thức.

Thông tin dự án

📅 Thời gian phát triển: 1 Tháng
🎮 Thể loại: Casual
📱 Nền tảng: Mobile (Android/iOS)
🛠️ Engine: Unity Engine
👥 Team Size: 1 Unity Dev, 1 Art, 1 Design
🏢 Studio: Quivering Studio
🔧 Design Pattern: Singleton, Object Pooling, Observer Pattern
💻 Development Tools: Visual Studio 2022, Rider
🔌 Plugins: Dotween

Trách nhiệm

  • Kết hợp với design tạo công cụ hỗ trợ xếp map.
  • Code core game play, các chức năng trong game.
  • Áp dụng Design Pattern: singleton, object pooling, observer pattern.
