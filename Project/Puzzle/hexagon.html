Giới thiệu dự án

Hexagon là game puzzle với gameplay chính di chuyển các element có màu giống nhau vào vị trí gần nhau để các màu merge lại và hoàn thành target của level. Người chơi cần chiến lược để merge các màu hiệu quả.

Game được phát triển trong 1 tháng, sử dụng công cụ xếp map tùy chỉnh để tạo level đa dạng.

Thông tin dự án

📅 Thời gian phát triển: 1 Tháng
🎮 Thể loại: Puzzle
📱 Nền tảng: Mobile (Android/iOS)
🛠️ Engine: Unity Engine
👥 Team Size: 1 Unity Dev, 1 Art, 1 Design
🏢 Studio: Quivering Studio
🔧 Design Pattern: Singleton, Object Pooling, Observer Pattern
🔌 Plugins: Dotween

Trách nhiệm

  • Kết hợp với design tạo công cụ hỗ trợ xếp map.
  • Code core game play, các chức năng trong game.
  • Áp dụng Design Pattern: singleton, object pooling, observer pattern.
