Giới thiệu dự án

Match Frenzy là game puzzle match 3 vui nhộn, nơi người chơi ghép các viên kẹo cùng màu để tạo combo, đạt điểm cao và hoàn thành các level đầy thách thức.

Game được phát triển trong 1 tháng, tập trung vào gameplay mượt mà và hiệu ứng hấp dẫn.

Thông tin dự án

📅 Thời gian phát triển: 1 Tháng
🎮 Thể loại: Puzzle / Match 3
📱 Nền tảng: Mobile (Android/iOS)
🛠️ Engine: Unity Engine
👥 Team Size: 1 Unity Dev, 1 Art, 1 Design
🏢 Studio: Quivering Studio
🔧 Design Pattern: Singleton, Object Pooling, Observer Pattern
🔌 Plugins: Dotween

Trách nhiệm

  • Kết hợp với design tạo công cụ hỗ trợ xếp map.
  • Code core game play, các chức năng trong game.
  • Áp dụng Design Pattern: singleton, object pooling, observer pattern.
